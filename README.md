# 0ch_plugins
0ch_plsingと打ちそうになったのは内緒

それでもわからない<ruby>場合<rp>(</rp><rt>ばあい</rt><rp>)</rp></ruby>は、<ruby>下<rp>(</rp><rt>した</rt><rp>)</rp></ruby>をお<ruby>読<rp>(</rp><rt>よ</rt><rp>)</rp></ruby>みください。

(このプラグインは<ruby>設定無<rp>(</rp><rt>せっていな</rt><rp>)</rp></ruby>しで<ruby>0ch_ownercmd.pl<rp>(</rp><rt>スレ主専用コマンド</rt><rp>)</rp></ruby>との<ruby>連携<rp>(</rp><rt>れんけい</rt><rp>)</rp></ruby>が<ruby>可能<rp>(</rp><rt>かのう</rt><rp>)</rp></ruby>です)

## 0ch_aa.pl
<ruby>AA<rp>(</rp><rt>アスキーアート</rt><rp>)</rp></ruby>をきれいに<ruby>表示<rp>(</rp><rt>ひょうじ</rt><rp>)</rp></ruby>することができる<ruby>HTML<rp>(</rp><rt>HyperText Markup Language</rt><rp>)</rp></ruby>タグ (&lt;aa&gt;〜&lt;/aa&gt;)を<ruby>追加<rp>(</rp><rt>ついか</rt><rp>)</rp></ruby>します。

(データは[ip-api.com](https://ip-api.com/)から<ruby>取得<rp>(</rp><rt>しゅとく</rt><rp>)</rp></ruby>しています。)

## 0ch_nanasiname.pl
<ruby>スレ主<rp>(</rp><rt>スレッド主</rt><rp>)</rp></ruby>が!774任意の名無し!3を<ruby>本文<rp>(</rp><rt>ほんぶん</rt><rp>)</rp></ruby>に<ruby>入力<rp>(</rp><rt>にゅうりょく</rt><rp>)</rp></ruby>することで<ruby>名無<rp>(</rp><rt>なな</rt><rp>)</rp></ruby>し<ruby>名<rp>(</rp><rt>めい</rt><rp>)</rp></ruby>を<ruby>変更<rp>(</rp><rt>へんこう</rt><rp>)</rp></ruby>することができます。
(このプラグインは<ruby>設定無<rp>(</rp><rt>せっていな</rt><rp>)</rp></ruby>しで<ruby>0ch_ownercmd.pl<rp>(</rp><rt>スレ主専用コマンド</rt><rp>)</rp></ruby>との<ruby>連携<rp>(</rp><rt>れんけい</rt><rp>)</rp></ruby>が<ruby>可能<rp>(</rp><rt>かのう</rt><rp>)</rp></ruby>です)

## 0ch_ownercmd.pl
スレッドを<ruby>建<rp>(</rp><rt>た</rt><rp>)</rp></ruby>てた<ruby>人<rp>(</rp><rt>ひと</rt><rp>)</rp></ruby>のみ<ruby>使用<rp>(</rp><rt>しよう</rt><rp>)</rp></ruby>できるコマンドを<ruby>実装<rp>(</rp><rt>じっそう</rt><rp>)</rp></ruby>するプラグインです。 <ruby>>>1<rp>(</rp><rt>スレ主</rt><rp>)</rp></ruby>と<ruby>同<rp>(</rp><rt>おな</rt><rp>)</rp></ruby>じホスト<ruby>名<rp>(</rp><rt>めい</rt><rp>)</rp></ruby>(または<ruby>端末識別子<rp>(</rp><rt>たんまつしきべつし</rt><rp>)</rp></ruby>)ならコマンドを<ruby>実行<rp>(</rp><rt>じっこう</rt><rp>)</rp></ruby>できます。 また、スレ<ruby>建<rp>(</rp><rt>た</rt><rp>)</rp></ruby>て<ruby>時<rp>(</rp><rt>じ</rt><rp>)</rp></ruby>にメール<ruby>欄<rp>(</rp><rt>らん</rt><rp>)</rp></ruby>に「!owner:パスワード:」と<ruby>入力<rp>(</rp><rt>にゅうりょく</rt><rp>)</rp></ruby>すると、ホスト<ruby>名<rp>(</rp><rt>めい</rt><rp>)</rp></ruby>が<ruby>変<rp>(</rp><rt>か</rt><rp>)</rp></ruby>わった<ruby>場合<rp>(</rp><rt>ばあい</rt><rp>)</rp></ruby>でもメール<ruby>欄<rp>(</rp><rt>らん</rt><rp>)</rp></ruby>に「!owner:パスワード:!stop」のように<ruby>入力<rp>(</rp><rt>入力</rt><rp>)</rp></ruby>することでコマンドを<ruby>実行<rp>(</rp><rt>じっこう</rt><rp>)</rp></ruby>できます。 ※パスワードの<ruby>後<rp>(</rp><rt>うし</rt><rp>)</rp></ruby>ろにもコロン「:」があります。  
  
※<ruby>スレスト機能<rp>(</rp><rt>スレッドストッパーきのう</rt><rp>)</rp></ruby>を<ruby>有効<rp>(</rp><rt>ゆうこう</rt><rp>)</rp></ruby>にするために、<ruby>0ch+<rp>(</rp><rt>ぜろちゃんねるプラス</rt><rp>)</rp></ruby>(0.7.4, 0.7.5)の<ruby>書<rp>(</rp><rt>か</rt><rp>)</rp></ruby><ruby>換<rp>(</rp><rt>か</rt><rp>)</rp></ruby>えが<ruby>必要<rp>(</rp><rt>ひつよう</rt><rp>)</rp></ruby>です。 module/vara.pl の sub Write の<ruby>最後<rp>(</rp><rt>さいご</rt><rp>)</rp></ruby>、235<ruby>行目<rp>(</rp><rt>ぎょうめ</rt><rp>)</rp></ruby>の return の<ruby>直前<rp>(</rp><rt>ちょくぜん</rt><rp>)</rp></ruby>に<ruby>以下<rp>(</rp><rt>いか</rt><rp>)</rp></ruby>の<ruby>行<rp>(</rp><rt>ぎょう</rt><rp>)</rp></ruby>を<ruby>加<rp>(</rp><rt>くわ</rt><rp>)</rp></ruby>えてください。
```

```
  
さらに、この<ruby>改造版<rp>(</rp><rt>改造版</rt><rp>)</rp></ruby><ruby>0ch_ownercmd.pl<rp>(</rp><rt>スレ主専用コマンド</rt><rp>)</rp></ruby>では、<ruby>主表示<rp>(</rp><rt>主表示</rt><rp>)</rp></ruby>も<ruby>可能<rp>(</rp><rt>かのう</rt><rp>)</rp></ruby>です。
